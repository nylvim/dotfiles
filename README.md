# My dotfiles

I use [confast](https://github.com/nylvim/confast) to manage my dotfiles. Path mappings can be found at `config.yaml` in case you want to copy them manually.

Custom scripts can be found inside the `scripts` directory. All of them should be POSIX-compliant, since I use `dash` as my `/bin/sh`. The `npath` script is used to call other scripts inside `scripts/nonpath`.

I use Arch, BTW.
